# LanBucket Website

Official website for LanBucket - Simple Local Network File Sharing

## About

This is the static website for LanBucket, a Windows application that enables easy file sharing between devices on your local network using peer-to-peer technology.

## Website Structure

- `index.html` - Main website page
- `style.css` - Styling and responsive design

## Features Highlighted

- Automatic device discovery
- Direct peer-to-peer transfers
- Drag and drop file sharing
- Local network only (no internet required)
- Progress tracking
- Native Windows application

## Local Development

To view the website locally, simply open `index.html` in your web browser.

## Deployment

This site is designed to be hosted on GitHub Pages. To deploy:

1. Push changes to the repository
2. GitHub Pages will automatically serve the site from the root directory
